Welcome to Wff Helper!

This is a tool to verify and format proofs in propositional logic. You can start a new proof at the top of the page or by clicking the logo in the top left.

This tool does NOT solve proofs for you. It DOES verify the steps you input are logically valid, and it DOES verify once you have finished the proof.

Example Usage

Example 1: Suppose we are proving ⊢ (a => (a v b)) using Direct Proof.

  • First, enter a, assumption
  • Then, enter a v b, 1, I1
  • You should see QED automatically appended to your proof to signify it is complete

Example 2: Proving ⊢ (x => x) using No Proof Method.

  • First, enter x v ~x, axiom
  • Then, enter ~x v x, 1, E10
  • Then, enter x => x, 2, E18
  • QED

Example 3: Proving ⊢ (x => x) using Proof by Contradiction.

  • First, enter ~(x => x), Assumption
  • Then, enter ~(~x v x), 1, E18
  • Then, enter ~~x ^ ~x, 2, E17
  • Then, enter x ^ ~x, 3, E15
  • Then, enter F, 4, E15
  • QED

Syntax

Where w is a valid wff, # is a number, and anything in [] is optional

  • Axiom: w v ~w, axiom
  • Hypothesis: w, hypothesis
  • Assumption: w, from gamma
  • Equivalence rule: w, #, E#
  • Inference rule: w, #, [#,] I#
